# Player Profile Images

Place player profile images in this folder with the following names:
- ayusha.jpg (or .png)
- dawa.jpg (or .png)
- rekha.jpg (or .png)
- sandesh.jpg (or .png)
- anuskar.jpg (or .png)
- utsab.jpg (or .png)

If a player image is not found, the default profile image will be used automatically.
